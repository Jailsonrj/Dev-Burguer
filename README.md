<h1>Dev Burguer</h1>

Sistema full stack de gestão de produtos e pedidos, desenvolvido com foco em boas práticas e arquitetura escalável. A aplicação permite cadastro, edição e listagem de produtos, upload de imagens, autenticação de usuários e filtragem dinâmica por categorias.

No front-end, utiliza React com gerenciamento de rotas, formulários e estado. No back-end, foi construída uma API REST com Node.js seguindo arquitetura MVC, integrada a banco de dados relacional e não relacional.

O projeto também conta com recursos como autenticação via JWT, criptografia de senhas, integração com pagamentos e containerização com Docker, simulando um ambiente próximo ao de aplicações reais em produção.
